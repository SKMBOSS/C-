1. 블록의 움직임
BLOCK클래스의 멤버변수 r과 c의 위치에 따라 바뀜.
/*block의 멤버함수*/
down() -> r의 값이 1증가
left() -> c의 값이 2 감소
right() -> c의 값이 2증가 

/*virtual func*/ 
set   ->현재 블록의 위치정보를 GameBoard에 전달
clear ->이전 블록의 위치정보를 0으로 초기화

isEdge ->바닥이 닿았는지 확인
+바닥충돌이 확인되면 edgecount 증가 edgecount가 2이상이라면 자리고정
->바닥충돌이 확인되었다가 없어졌다면 edgecount를 초기화 


isEdgeCrash ->옆블록 (+경게선)에 충돌이 되는지 확인
현재위치에 정보가 진행이 불가능한 곳일 때 이전값으로 되돌리는지 확인 


drop ->블록을 떨어뜨림
down을 반복, isEdge를 이용해 자리고정확인

2. 블록의 회전
Block의 멤버변수 rotateForm (초기에 0)
한번회전할수록 rotate를 증가
해당 회전수에 맞는 set clear isEdge...등의 정보를 가지고오게함
-> 회전이 불가능한지 확인-> 미리회전한 블록이 다른 영역을 침범하면 회전을 하지못하고, 그렇지 않다면 회전한 값을 유지
